# API Reference

Welcome to the Dripr API documentation. Our API provides access to real estate market data and analytics.

## Available APIs

### Version 1

- [Local Market Data API](v1/local-market-data.html) - Get real estate market information for specific zip codes, including pricing trends, inventory levels, and market statistics.

## Getting Started

All API endpoints require authentication via API key. Contact [Dripr](https://www.dripr.ai/) to obtain your API key.

Include your API key in the request headers:

```
X-API-Key: your_api_key_here
```

## Base URL

```
https://api.dripr.ai
```

## Support

For questions or issues with the API, please contact Dripr support.
